Dynamic Table

Description
  • On pressing Add Record, a new record will be created.
  • Clicking the select button and then clicking on a row or the field will cause the entity to be selected.
  • On selection and clicking the delete button will cause the row or the field to be removed.
  • Clicking on the add new field, textboxes will be created in the table where any input can be given and on pressing enter key the new field will be added.
